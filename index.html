📐 SimpleCAD の使い方

ツール（キー）

  • 選択 V / 線 L / 矩形 R / 円 C / 楕円 O / 連線 P / 文字 T / 寸法 M / 角度 A / 削除 E
  • 角度寸法: 頂点→1辺目→2辺目の3クリック(プロパティ欄で内角を数値編集)。連続寸法(連寸)は点を連続クリックし確定で隣接区間を一括寸法化
  • 文字は選択中にプロパティ欄で左/中央/右に整列
  • 画面フィット F / ヘルプ ?
  • 正多角形は左パレットの ⬡（辺数は右パネル）

操作

  • スマホ: 2本指でピンチ=ズーム/ドラッグ=パン、1本指で作図
  • PC: ホイール=ズーム、スペース+ドラッグ or 中ボタン=パン
  • 複数選択: 空き場所をドラッグ、Shift+クリックで追加、Ctrl+Aで全選択
  • 変形: 単一選択で四隅=リサイズ、上の丸=回転（Shiftで直交/正方形、吸着ONで15°）
  • Alt+ドラッグで複製しながら移動

ショートカット

  • 取消/やり直し Ctrl+Z / Ctrl+Y
  • コピー/切取/貼付/複製 Ctrl+C / X / V / D
  • 移動 矢印キー（Shiftで10倍）、削除 Delete

その他

  • レイヤー / 整列・分布 / 重ね順 / 数値直接入力 に対応
  • 保存・読込(JSON)、DXF/SVG取り込み(📥取込)、画像トレース、SVG / PNG / PDF(実寸) 書き出し
  • 実寸印刷(🖨 書出メニュー / Ctrl+P。印刷時は倍率100%を選択)
  • ホーム画面に追加でオフライン利用可（PWA）
