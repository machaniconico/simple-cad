# SimpleCAD — ブラウザ簡易CAD

スマホでもPCでも使える、依存ライブラリなしの**単一HTMLファイル**製2D CADです。
`index.html` をブラウザで開くだけで動きます（インストール・サーバー不要）。

## 使い方

`index.html` をダブルクリック、またはブラウザにドラッグして開くだけです。
スマホでは、ファイルをクラウド/ローカルに置いてブラウザで開いてください。

## 操作

### ツール（左パレット）
| アイコン | 機能 |
|---|---|
| 🖱 選択 | 図形をタップで選択、ドラッグで移動。単一選択時は回転・リサイズハンドル表示。 |
| ／ 線 | 始点ドラッグ終点で線分。長さをmm表示。 |
| ▭ 矩形 | ドラッグで矩形。幅×高さをmm表示。 |
| ◯ 円 | 中心からドラッグで円。半径をmm表示。 |
| ⏚ 連線 | タップ連打でポリライン。始点付近で閉じる／「確定」ボタン／ダブルタップで終了。 |
| A 文字 | タップで入力欄が開き、テキスト注記を配置。既存テキストはダブルクリックで編集。 |
| ↔ 寸法 | 2点間の距離をmm注記として配置。 |
| ⌫ 削除 | タップした図形を削除。 |

### ビュー操作
- **スマホ**: 2本指でピンチ→ズーム、2本指ドラッグ→パン。1本指は作図。
- **PC**: マウスホイールでズーム、スペース＋ドラッグ（または中ボタン）でパン。

### 変形（選択ツール・単一選択時）
- **リサイズ**: 四隅の□ハンドルをドラッグ。
- **回転**: 上部の●ハンドルをドラッグ（グリッド吸着ON時は15°スナップ）。
- **数値直接入力**: 右パネルに選択図形の座標・寸法・角度・テキスト本文が出て、直接編集できます。

### レイヤー（🗂 レイヤー）
- レイヤーの追加／リネーム／表示・非表示（👁）／ロック（🔒）／アクティブ切替／削除。
- 新規図形はアクティブレイヤーに作成。選択図形を現レイヤーへ移動（⇲）。
- 非表示レイヤーは描画・書き出し・選択の対象外。ロックレイヤーは編集不可。

### その他
- **グリッド吸着 / 端点スナップ**: 右パネルでON/OFF、グリッド間隔(1〜100mm)変更。
- **線色・塗り・線幅・字高**: 右パネルで設定（新規作成および「選択へ適用」）。
- **Undo/Redo**: ツールバー、または `Ctrl+Z` / `Ctrl+Y`。`Delete`キーで選択削除。
- **保存/読込**: JSONでローカル保存・復元。さらにブラウザに自動保存され、次回開くと復元されます。
- **書き出し**: SVG（実寸mm・ベクター）/ PNG / **PDF（実寸mm）** をダウンロード。
- **Fit**: 全図形が収まるようズーム。**New**: 全消去。

## 座標系
- 単位はミリメートル(mm)。ステータスバーにカーソル座標・倍率・グリッド間隔・図形数を表示。
- 倍率100% = 2px/mm。

## 技術メモ
- 純粋な HTML + CSS + Canvas 2D + Pointer Events。外部依存ゼロ（ランタイム）。
- 高DPI(`devicePixelRatio`)対応。`touch-action: none` でブラウザ既定ジェスチャを抑止。
- 回転は各図形の `rot`(ラジアン)で保持し、ヒットテスト/ハンドルは中心まわり逆回転でローカル座標化。
- PDFは依存なしで生成：図面をSVG化→高解像度ラスタライズ→`FlateDecode`で可逆埋め込み、ページサイズを実寸mm（ポイント換算）に設定。日本語テキストも崩れません。
- 外部JSON/localStorage由来データはサニタイズ（型allowlist・色検証・`__proto__`除去）してから取り込み。

## 検証
ヘッドレスブラウザ（Playwright/Chromium）による自動テストを同梱しています。

```bash
npm install            # playwright を取得（初回のみ）
npx playwright install chromium
node test/verify.mjs   # 機能テスト 65項目
node test/pdfcheck.mjs # 生成PDFの構造検証（xrefオフセット厳密）
```

- 作図・選択移動・スナップ・Undo/Redo・保存読込・SVG/PNG/PDF書き出し・ピンチズーム・レイヤー・文字・変形（回転/リサイズ）・入力サニタイズまで計65項目が全パスします。
- コードレビュー／セキュリティレビューを実施し、指摘事項は反映済みです。
